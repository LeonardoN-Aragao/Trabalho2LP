# Trabalho2LP

## Como jogar

Para rodar o projeto precisa baixar e instalar:  
- GHCI
- Cabal-Install

link para download: https://www.haskell.org/downloads/

No terminal utilze os comandos:
$ cabal-install update
$ cabal-install random
$ ghci
$ :cd /{WorkDiretory}
$ :l main.hs
$ main
